# 三个数据集迁移性与隐蔽性的论文导向分析

本报告只讨论 `NC` 之外的主结果。迁移性使用目标域测试 `transfer_rate`，隐蔽性使用四种防御 `STRIP / SentiNet / SCaLe-Up / IBD-PSC` 的平均逆指标 `stealth_mean`，综合折中指标使用 `tradeoff_hmean`。这里的重点不是简单给出排名，而是回答论文主题本身：哪些攻击真正同时体现了“可迁移”和“难检测”，哪些方法只能证明其中一面，哪些异常结果反而能增强论文论证。

## 1. 最核心的结论

1. 单看迁移性，`basic`/BadNet 与部分 `belt` 配置几乎总能给出很高的目标域 ASR，但它们的隐蔽性很差，因此更适合作为“高迁移但不隐蔽”的对照组，而不是论文中“兼顾迁移与隐蔽”的主证据。
2. 真正能支撑论文主题的主力方法不是局部 patch，而是分布式或结构性触发器：`blend`、`adaptive_blend`、`WaNet`，以及在部分数据集上表现出高上限的 `upgd` 与 `SIG`。
3. 三个数据集的最优方法并不相同，说明“迁移性”不是攻击方法的绝对属性，而是由触发器形式、源域/目标域差异、以及防御机制共同决定的。这个结论本身就能支撑你的论文主题。

## 2. 三个数据集的总体判断

| 数据集 | transfer_mean | stealth_mean_mean | 结论 |
|---|---:|---:|---|
| CIFAR-10 -> STL-10 | 0.5927 | 0.4461 | 迁移性中等偏强，但防御更容易抓到局部触发器，适合展示“高迁移不等于高隐蔽” |
| Tiny-ImageNet -> Target Domain | 0.6697 | 0.4618 | 目标域迁移总体最强，更适合展示结构型/分布式触发器如何兼顾迁移与隐蔽 |
| MNIST-M -> MNIST | 0.6031 | 0.5627 | 平均隐蔽性最高，但结果分化也最大；这里最适合讨论“真隐蔽”和“触发器失效导致的伪隐蔽” |

从论文写作角度，这三个数据集分别承担不同角色：

- `CIFAR-10 -> STL-10` 适合证明：局部 patch 虽然迁移强，但非常不隐蔽。
- `Tiny-ImageNet -> Target Domain` 适合证明：空间形变与全局混合触发更能在自然图像域移位下保持“迁移-隐蔽”折中。
- `MNIST-M -> MNIST` 适合证明：有些方法看起来很隐蔽，其实是目标域根本没触发起来，典型就是 `SIG`。

## 3. 攻击方法的特定分析

### 3.1 `basic` / BadNet

- `basic` 在三个数据集上的平均迁移率都很高：`0.9093 / 0.8147 / 0.8626`，说明它非常适合证明“后门可以迁移”。
- 但它的平均隐蔽性是所有方法里最差或接近最差：`0.1256 / 0.1835 / 0.3139`，对应平均折中分只有 `0.1736 / 0.2213 / 0.3691`。
- 这说明 `basic` 的论文价值不在于“隐蔽性”，而在于提供一个强烈对照：高迁移率并不自动意味着高隐蔽性。
- `SentiNet` 对它几乎是定点打击，TPR 分别达到 `0.9887 / 1.0000 / 0.9199`，这和局部 patch 的方法机制完全一致。

### 3.2 `adaptive_patch`

- `adaptive_patch` 比 `basic` 有更好的折中，但仍然保留局部触发器特征。平均折中分为 `0.4275 / 0.3164 / 0.4910`。
- 在 `CIFAR-10 -> STL-10` 和 `MNIST-M -> MNIST` 上，它确实比 `basic` 更隐蔽，但在 `Tiny-ImageNet -> Target Domain` 上仍被 `SentiNet` 完全抓住，TPR=`1.0000`。
- 因此它可以写成“对 patch 攻击的改进”，但不适合作为你论文中“最强隐蔽方法”的代表。

### 3.3 `belt`

- `belt` 是一个非常有代表性的中间形态：它比 `basic` 更隐蔽，但仍然保留局部区域与 mask/cover 的几何特征。
- 在 `MNIST-M -> MNIST` 上它是平均折中最好的方法，`tradeoff_hmean_mean=0.5822`，同时迁移率 `0.7466`、隐蔽性 `0.6104` 都不低，说明这种“局部但经过 cover/mask 处理”的设计在手写数字域迁移里非常有效。
- 但在 `Tiny-ImageNet -> Target Domain` 上，它虽然平均迁移率最高 `0.8867`，隐蔽性却只有 `0.3074`，说明自然图像目标域里，这类局部触发更容易暴露。
- 因此 `belt` 最适合写成：在简单结构域中可以兼顾迁移与隐蔽，但在复杂自然图像域中更偏“高迁移、低隐蔽”。

### 3.4 `blend`

- `blend` 是三个数据集上最稳定、最适合当论文主证据的方法之一。平均折中分分别是 `0.4732 / 0.5076 / 0.5417`，没有明显短板。
- 它在 `MNIST-M -> MNIST` 上尤其稳定，迁移率 IQR 只有 `0.0100`，但平均迁移率仍达到 `0.8466`。这说明它不是靠少数极端配置，而是整体上都能工作。
- 在 `Tiny-ImageNet -> Target Domain` 上它的平均迁移率 `0.7125`、隐蔽性 `0.5373`，整体仅次于 `WaNet`，是自然图像目标域里非常强的折中方法。
- 如果你的论文要强调“真正稳健的迁移性与隐蔽性兼顾”，`blend` 应该是主角之一。

### 3.5 `adaptive_blend`

- `adaptive_blend` 的模式和 `blend` 很像，但更偏向隐蔽性。平均折中分分别为 `0.4340 / 0.4869 / 0.5316`。
- 在 `MNIST-M -> MNIST` 上，它的平均隐蔽性 `0.6374` 高于 `blend` 的 `0.5018`，而迁移率仍有 `0.6251`，说明自适应混合确实在防御规避上有价值。
- 在 `Tiny-ImageNet -> Target Domain` 上，它也优于大多数 patch 类方法，说明“把触发分散到整幅图像”在目标域迁移时更不容易被局部解释型防御抓住。
- 论文中可以把 `adaptive_blend` 写成：比普通 `blend` 更偏向隐蔽性优化的版本。

### 3.6 `SIG`

- `SIG` 是最需要单独讨论的特殊方法。它在三个数据集上的平均隐蔽性都是最高：`0.5990 / 0.7026 / 0.7027`。
- 但它不是在所有数据集上都“真正隐蔽”。在 `MNIST-M -> MNIST` 上，它的平均迁移率只有 `0.0026`，平均 ASR 也只有 `0.0279`，折中分仅 `0.0051`。这说明这里的高隐蔽不是防御没抓到，而是触发器在目标域几乎失效了。
- 在 `CIFAR-10 -> STL-10` 上，`SIG` 反而是很强的平衡方法，平均折中分 `0.4484`，并且最佳单点配置达到 `tradeoff_hmean=0.7709`。
- 在 `Tiny-ImageNet -> Target Domain` 上，`SIG` 的方法均值并不高，`tradeoff_hmean_mean=0.3074`，但最佳 `resnet18` 配置可达到 `0.7781` 和 `0.7730`。这说明 `SIG` 不是天然弱，而是高度依赖参数与架构匹配。
- 因此 `SIG` 在论文里不能简单写成“高隐蔽攻击”，更准确的表述应该是：`SIG` 具有很强的防御规避能力，但它的跨域迁移性强依赖目标域是否保留原始频率结构。

### 3.7 `WaNet`

- `WaNet` 是 `Tiny-ImageNet -> Target Domain` 上平均折中最好的方法，`tradeoff_hmean_mean=0.5985`，同时迁移率 `0.7251`、隐蔽性 `0.5982` 都高。
- 在 `MNIST-M -> MNIST` 上它也很强，平均折中分 `0.5311`，而且 `vgg` 上的平均折中甚至达到 `0.6607`。
- 但在 `CIFAR-10 -> STL-10` 上它明显掉队，平均迁移率只有 `0.1951`，折中分 `0.2190`。
- 这组结果非常有论文价值，因为它说明空间形变型后门不是普遍最优，而是在“几何结构能够跨域保留”的场景里最强，在 `Tiny-ImageNet` 和 `MNIST-M` 上都成立，在 `CIFAR-10 -> STL-10` 上则不成立。

### 3.8 `upgd`

- `upgd` 是最典型的“高上限、低稳定性”方法。
- 在 `CIFAR-10 -> STL-10` 上，它的平均折中分最高，`tradeoff_hmean_mean=0.5321`，说明在这个任务上它不仅能打，而且整体稳。
- 但在 `Tiny-ImageNet -> Target Domain` 与 `MNIST-M -> MNIST` 上，它的方法均值只有 `0.3751` 和 `0.3588`，看起来并不突出。
- 问题在于它的方差极大。`MNIST-M -> MNIST` 上它的迁移 IQR 高达 `0.9090`，`Tiny-ImageNet -> Target Domain` 上也有 `0.6709`，说明不同参数配置之间差异极大。
- 与此同时，它的最佳单点配置又极强：`tiny_imagenet` 上最佳 `tradeoff_hmean=0.7944`，`mnistm` 上最佳 `tradeoff_hmean=0.8539`，都是全局最强级别。
- 所以 `upgd` 最准确的论文定位不是“平均最优”，而是“最有潜力但最依赖调参”的高天花板攻击。

## 4. 防御方法的特定分析

### 4.1 `SentiNet`

- `SentiNet` 的结果最容易解释，因为它本质上就是局部解释型检测。
- 它对局部 patch 攻击极强：`basic`/`adaptive_patch` 在 `cifar10` 上的 TPR 是 `0.9887 / 0.9371`，在 `tiny_imagenet` 上都是 `1.0000`，在 `mnistm` 上也有 `0.9199 / 0.8004`。
- 但它对非局部攻击明显无力，例如 `mnistm` 上 `blend / adaptive_blend / SIG / WaNet / upgd` 的 TPR 只有 `0.1145 / 0.0759 / 0.0868 / 0.0777 / 0.0700`。
- 因此，`SentiNet` 的实验结果不是在证明“这些方法绝对隐蔽”，而是在证明：当触发器不再集中于一个显著局部区域时，局部解释型防御会系统性失效。

### 4.2 `STRIP`

- `STRIP` 总体上是四种防御里最弱的一类，尤其不擅长处理 `SIG`、`WaNet`、`blend` 这类非局部触发。
- 典型例子是：`cifar10` 上 `WaNet` 的 TPR 只有 `0.0306`，`mnistm` 上 `SIG` 的 TPR 只有 `0.0506`，`tiny_imagenet` 上 `WaNet` 的 TPR 也只有 `0.0664`。
- 这说明当触发模式分散到全局纹理、频率结构或几何形变中时，简单的输入混合熵检测很难维持判别力。

### 4.3 `SCaLe-Up`

- `SCaLe-Up` 的表现最依赖触发器在强度缩放后是否仍稳定主导分类。
- 在 `cifar10` 上，它对 `basic`、`upgd`、`belt` 的检测较强，TPR 分别为 `0.7936 / 0.7066 / 0.6929`，说明对强驱动触发器更敏感。
- 但在 `tiny_imagenet` 上它对 `SIG` 和 `upgd` 几乎无能为力，TPR 只有 `0.0532 / 0.0497`。
- 反过来，在 `mnistm` 上它对 `WaNet` 却是最强防御，TPR=`0.6111`。这说明空间扭曲在该数据域中会诱发明显的尺度一致性异常，这是一个值得在论文里单独点出的反常结果。

### 4.4 `IBD-PSC`

- `IBD-PSC` 是四种防御里覆盖面最宽、总体最强的通用防御。三数据集平均 TPR 分别为 `0.5199 / 0.4920 / 0.5796`，都是同数据集里的最高值。
- 它往往是 `belt`、`blend`、`WaNet`、`SIG`、`upgd` 这些“非典型局部 patch”攻击的最强检测器。
- 因此，如果一个方法在 `IBD-PSC` 下仍然保持高 `stealth_mean` 和高 `transfer_rate`，它才更有资格被写成真正兼顾迁移与隐蔽的攻击。

## 5. 这份结果如何直接支撑论文主题

如果你的论文主题是“证明后门攻击具有迁移性和隐蔽性”，那么最有说服力的写法不是把所有方法一视同仁，而是把它们分成三类：

- `high-transfer but low-stealth`：
  `basic`、部分 `belt`、部分 `adaptive_patch`。它们证明后门可以跨域迁移，但不能证明隐蔽性。
- `balanced evidence`：
  `blend`、`adaptive_blend`、`WaNet`。它们在多个数据集上同时保持中高迁移率和中高隐蔽性，是论文主线最稳的证据。
- `high-ceiling but unstable / domain-sensitive`：
  `upgd`、`SIG`。它们说明某些攻击可以达到非常强的“迁移-隐蔽”上限，但这种优势并不稳定，依赖参数、架构和域差异。

按数据集给出最推荐的主论证组合：

- `CIFAR-10 -> STL-10`：
  主写 `upgd + blend + SIG`，对照写 `basic`。这一组最能说明“局部 patch 虽然迁移强，但真正兼顾隐蔽性的还是分布式/优化型触发器”。
- `Tiny-ImageNet -> Target Domain`：
  主写 `WaNet + blend + adaptive_blend`，补充 `upgd` 和 `SIG` 的高上限配置。这里最能说明“自然图像目标域中，结构型和分布式触发器更适合兼顾迁移与隐蔽”。
- `MNIST-M -> MNIST`：
  主写 `belt + blend + WaNet + adaptive_blend`，明确把 `SIG` 当成失败案例。这里最能说明“高隐蔽性必须建立在目标域仍能触发后门的前提上”。

## 6. 论文中最值得直接写进去的三句话

1. 局部 patch 攻击能够稳定证明跨域迁移性，但难以证明隐蔽性，因为它们会被 `SentiNet` 与 `IBD-PSC` 显著检测到。
2. 在三个数据集上，真正稳定支撑“迁移性-隐蔽性折中”的主力方法是 `blend`、`adaptive_blend` 和 `WaNet`；`upgd` 与 `SIG` 则表现出更高的上限，但更强的参数敏感性和域依赖性。
3. `MNIST-M -> MNIST` 上 `SIG` 的结果表明，高隐蔽性不能脱离迁移性单独解释，否则容易把“目标域触发失效”误判为“成功绕过检测”。
